# 🧬 DNA to Protein Translator

This is a simple bioinformatics Python project that:

- Detects the first start codon (ATG)
- Translates DNA into protein sequence
- Stops at stop codon (TAA, TAG, TGA)
- Prints the final protein sequence

## Example

Input:
CCCATGTTTGAACTGAAATAGGGT

Output:
MFELK

## How to Run

python dna_translator.py
